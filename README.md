# 🎬 Senema – Movie Search Engine

Senema is a modern and responsive Movie Search Engine built using HTML, CSS, and JavaScript. It provides users with an interactive platform to explore movies, view detailed information, watch trailers, and create a personal watchlist. The project focuses on delivering a clean user interface, smooth user experience, and dynamic movie browsing functionality.

## Features

* Modern and responsive user interface
* Search movies instantly
* Browse popular movies
* Detailed movie information modal
* Movie poster, genre, release year, and description
* Watch trailer functionality
* Add movies to a personal watchlist
* Dark mode toggle
* Mobile-friendly design
* Interactive animations and hover effects
* Pure frontend implementation using JavaScript

## Technologies Used

## How It Works

1. Users can search for movies using the search bar.
2. Movie cards are dynamically displayed on the homepage.
3. Clicking on a movie opens a detailed modal window.
4. Users can view movie information such as:

   * Title
   * Release Year
   * Genre
   * Description
5. Users can watch trailers through the trailer button.
6. Movies can be saved to a personal watchlist.
7. Dark mode provides an alternative viewing experience.
